# vast-adapter
RecyclerView通用适配器插件
1，满足RecycleView中LinearLayoutManager、GridLayoutManager、StaggeredGridLayoutManager三种布局管理器的UI绑定；
2，封装了对HeaderView、FooterView、点击事件、长按事件的支持，同时兼容上述三种布局管理器；
3，支持对缺省图的设置，可使用默认样式进行设置，也可完全自定义View视图进行设置。
